# Alibi UI Guardrails

Use this as the default reference point before adding new screen chrome.

## Baseline

- Prefer plain layout over expressive layout.
- One strong action area per screen.
- One card treatment only: subtle border, low contrast surface, standard radius.
- Use accent color only for actions, not for large decorative blocks.

## Do not add by default

- hero banners
- stacked accent panels
- decorative chips without real state meaning
- multiple visual focal points above the fold
- explanatory copy longer than 2 short lines

## Screen structure

1. title
2. one-line orientation
3. primary action block
4. plain sections for everything else

## Tone

- restrained
- functional
- confident
- not ornamental

If a screen feels "designed," simplify it once more.
